micro.css

version 0.1a

About:

Micro.css is developed in an ever growing sea of css frameworks to be different. Its key difference is its size. Not everyone needs 10 font styles, tetrad colour scheme and 16 breakpoints from the get go. You can add all the fluff later, you need the basics to work right now. That is where micro.css excels at. Having typography, basic grid, forms, buttons and tables solved for you in a miniscule package under 2 kilobytes.

Built with love and coffee.

Typography:

Introducing automatical scaling of all typography through clever coding. The code itself can be found on various websites in one form or another, but it is implemented from the start. And if you don't like it you can remove it simply by removing first few lines of css.

Try resizing your browser to see the effect first hand.

Grid:

The grid itself is comprised out of 12 columns. Due to practicality and my personal biases it might be cut down to 6 columns.

All you need is a class="column [number]" where [number] represents number of columns from one to twelve
